
# Black Jack - Capstone Project

Welcome to BlackJack! This Python project simulates the classic game of Black Jack! The program provides a simple command-line interface for user interaction.


## Features

- Realistic Gameplay with Accurate Rules
- Random Card Dealing 
- Dealer AI
- Score Tracking 
- User-friendly Console Interface
- Time Delays for Better Gaming Experience
- Ascii Art for Choice Visualization

## Requirements

- Python 3.6 or higher
- Terminal Emulator
- Text editor or IDE 
  
## Installation

1. Download or clone the repository:

```sh
  git clone https://github.com/RiyaBodke/100_Days_of_Code_The_Complete_Python_Pro_Bootcamp
```
or download the ZIP file and extract it.

2. Navigate to the Project Directory:

```sh
  cd 100_Days_of_Code_The_Complete_Python_Pro_Bootcamp
```

3. Navigate to the Project Sub-Directory:

```sh
  cd P_11_BlackJack
```

4. Run the game:

```sh
   python main.py
```

5. Follow the on-screen instructions:

- Choose an option from 1 or 2 to select your move (hit or stand). Entering any number apart from 1 to 3 will result in an error and the program will prompt you to choose a valid option

## Rules

1. The cards have the following values:

- Number cards (2-10): Face value
- Face cards (Jack, Queen, King): 10 points
- Ace: 1 or 11 points (calculated as per the best outcome for the player)

2. This game can be played by two players - the user and the dealer.

3. Each player receives two cards. One of the dealer's cards is face-up and the other is revealed later. The score of face values of the total cards is tracked for both players.

4. The players should choose in such a way that the value of their cards doesn't exceed 21.

5. User's Turn:
The user has two options -
- Hit: Ask for another card.
- Stand: Stop drawing cards.

6. The dealer will have his turn after the player's turn.
